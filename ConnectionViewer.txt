■はじめに

  Connection Viewer は、TCP と UDP の接続状況を表示するプログラムです。
  IPv4 と IPv6 に対応しています。

  主に以下の機能があります。

  ・接続状況の表示
  ・接続ログの表示
  ・送受信速度のグラフ表示
  ・ネットワーク・インターフェースの状況表示
  ・Whois 情報の取得や GeoIP による地域表示
  ・特定アドレスの通信のブロック


■動作環境

  Windows Vista/7

  Vista で使用するには、SP2 以降に「Windows Vista 用のプラットフォーム更新プロ
  グラム(KB971644)」が適用されている必要があります(Windows Update で提供されて
  います)。


■ファイル構成

  ・ConnectionViewer.exe  プログラム本体
  ・ConnectionViewer.txt  説明文(このファイル)
  ・GeoIPDownloader.exe   GeoIP データベースのダウンローダ
  ・License.txt           ライセンス(GPL)

  プログラムの実行時に本体と同じフォルダに、設定を保存するために
  ConnectionViewer.ini ファイルが作成されます。


■インストール

  適当なフォルダにファイルを展開してください。
  設定の保存などのために、書き込み権限のあるフォルダに配置してください。

  GeoIP のデータベースファイルがあれば、各アドレスの国と地域を表示することがで
  きます。
  GeoIP のデータベースは以下のページからダウンロードできます。

  ・GeoLite City (国と地域を表示したい場合)
      http://www.maxmind.com/app/geolitecity

  ・GeoLite Country (国のみ)
      http://www.maxmind.com/app/geolitecountry

  上記のページで "Download the latest GeoLite City(Country) Binary Format" の
  リンクからダウンロードしてください。
  ファイルは gzip で圧縮されていますので、適当なアーカイバなどで展開してくださ
  い。展開したファイルをプログラムと同じフォルダに入れるか、設定でパスを指定す
  ると利用できます。

  添付の GeoIPDownloader.exe を使用すると、ダウンロードから展開までを自動で行え
  ます。GeoIP Downloader について詳しくは「■GeoIP Downloader について」の項を
  参照してください。

  なお、データベースは一か月に一回更新されますので、なるべく最新のものを利用し
  てください。


■アンインストール

  ファイルを削除してください。レジストリは使用していません。


■使い方など

  ・以下の機能を利用するためには、プログラムを管理者として実行する必要がありま
    す。

    ・各接続ごとの送受信速度や送受信量の表示
    ・通信のブロック
    ・接続の切断

  ・接続の切断が行えるのは IPv4 の TCP のみです。

  ・グラフとステータスバーの送受信速度/送受信量は、デフォルトで全てのハードウェ
    ア・インターフェースの合計になっています。
    特定のインターフェースの数値を表示したい場合、グラフを右クリックして表示さ
    れるメニューから選択してください。

  ・グラフ表示で左上の表示をクリックして、各項目の表示/非表示を切り替えることが
    できます。

  ・表のソートの比較基準の優先順位はカラムをクリックした順になります。
    例えば、新しくカラムをクリックした時、今まで優先順位が一番高かったものが2番
    目になります。


■GeoIP Downloader について

  GeoIP Downloader は、GeoIP のデータベースファイルをダウンロードするためのプロ
  グラムです。
  プログラムと同じフォルダにファイルがダウンロードされ、展開されます。その際、
  既存のファイルは上書きされます。
  URL は変更される可能性がありますので、その際は新しい URL を入力してください。

  動作としてはダウンロードと展開を行うだけのプログラムですので、代わりに wget
  と gzip を使うバッチファイルなどを使用してもいいでしょう。
